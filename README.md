# My Portfolio

Portfolio personal desarrollado con React, TypeScript, Vite y Tailwind CSS.

## Descripcion

Este sitio presenta mi perfil profesional como desarrollador Full Stack, incluyendo informacion sobre mi experiencia, habilidades tecnicas, proyectos destacados y un formulario de contacto.

## Tecnologias

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Lucide React
- React Typed

## Secciones

- Inicio
- Sobre mi
- Habilidades
- Proyectos
- Contacto
